    Subsection 2.7.1 Defining the Correlation

    Suppose we have two sets of numbers:
    \begin{equation*} {1,2,5,6,12} \end{equation*}
    \begin{equation*} {2,4,10,12,24} \end{equation*}
    If someone told you that the next number in the first set was 30, what would you expect the next number in the second set to be? You would probably guess 60, because thereâ€™s a clear pattern here - the number in the second series is always exactly double the corresponding number in the first series.
    ðŸ”—
    In practice, when we look at two sets of data, the relationship between them is rarely this exact. For example, look at these two sets of data:
    \begin{equation*} {1,2,5,6,12} \end{equation*}
    \begin{equation*} {2,5,9,14,21} \end{equation*}
    The values in the second set are just a little bit off from two times as much as the first set. If someone told you the next number in the first set was 30, though, you would still probably expect the next number in the second set to be approximately 60 - maybe not exactly 60, but something close. Even though one number doesnâ€™t exactly predict the other, it comes pretty close.
    ðŸ”—
    The correlation is a way of measuring how closely one set of numbers predict another. The correlation gives a number between 1 and -1. A correlation of 0 means that there is absolutely no relationship between the two sets of numbers - when one changes, you canâ€™t predict what will happen to the other. A correlation of 1 means that there is a perfect relationship between the numbers - when the values in the first data set increase, the values in the second data set increases in a way that exactly corresponds. Our first two sets of numbers would have a correlation of 1. Our second pair would have a correlation close to 1, but not exactly 1. A correlation of -1 also indicates a perfect relationship, but when one of the values in one set increases, the values in the second set will decrease in a way that exactly corresponds. So the pattern is still perfect, but it works in reverse.

    Subsection 2.7.2 Computing Correlation

    Computing the correlation is a little complicated, and is easiest done in multiple steps. While itâ€™s not crucial that you can go through all of these steps on your own, weâ€™ll explain how the correlation is defined to help understand why it computes how related two sets of data are.
    ðŸ”—

    Correlation.

    If we have two sets of data \(x = {x_1,x_2,\ldots,x_n}\) and \(y = {y_1,y_2,\ldots,y_n}\) with the same number of values, the correlation is
    \begin{equation*} cor(x,y) = \frac{(x_1-\overline{x})(y_1-\overline{y})+(x_2-\overline{x})(y_2-\overline{y})+\cdots+(x_n-\overline{x})(y_n-\overline{y})}{(n-1)s_x s_y} \end{equation*}
    where \(\overline{x}\) and \(\overline{y}\) are the means of \(x\) and \(y\text{,}\) and \(s_x\) and \(s_y\) are the sample standard deviations of \(x\) and \(y\text{.}\)
    ðŸ”—
    You can compute the mean by adding up all the values and dividing by \(n\text{.}\)
    \begin{equation*} \overline{x} = \frac{x_1+x_2+\cdots + x_n}{n} \end{equation*}
    ðŸ”—
    You can compute the sample standard deviation by subtracting the mean from each value, squaring that, adding them all up, dividing by \((n-1)\text{,}\) and taking the square root.
    \begin{equation*} s_x = \sqrt{\frac{(x_1 - \overline{x})^2 + (x_2 - \overline{x})^2 + \cdots + (x_n - \overline{x})^2}{(n-1)}} \end{equation*}
    ðŸ”—
    ðŸ”—
    This is a pretty complicated formula, but it turns out itâ€™s pretty easy to break it down into steps.
    1. Compute the means \(\overline{x}\) and \(\overline{y}\text{.}\)
      ðŸ”—
    2. Compute the sample standard deviations \(s_x\) and \(s_y\text{.}\)
      ðŸ”—
    3. Subtract the mean \(\overline{x}\) from each value in \(x\text{,}\) and the mean \(\overline{y}\) from each value in \(y\text{.}\)
      ðŸ”—
    4. Multiply the corresponding differences from \(x\) and \(y\) together.
      ðŸ”—
    5. Add them all up.
      ðŸ”—
    6. Divide the whole thing by \((n-1)s_x s_y\text{.}\)
      ðŸ”—
    ðŸ”—
    Letâ€™s see how this works with the second data set above:
    ðŸ”—
    \begin{equation*} x = {1,2,5,6,12} \end{equation*}
    \begin{equation*} y = {2,5,9,14,21} \end{equation*}
    First weâ€™ll compute the means and sample standard deviations of each data set:
    ðŸ”—
    \begin{equation*} \overline{x} = \frac{1+2+5+6+12}{5} = 5.2 \end{equation*}
    \begin{equation*} \overline{y} = \frac{2+5+9+14+21}{5} = 10.2 \end{equation*}
    \begin{equation*} s_x = \sqrt{\frac{(1-5.2)^2 + (2-5.2)^2 + (5-5.2)^2 + (6-5.2)^2 + (12-5.2)^2}{5-1}} = 4.3244 \end{equation*}
    \begin{equation*} s_y = \sqrt{\frac{(2-10.2)^2 + (5-10.2)^2 + (9-10.2)^2 + (14-10.2)^2 + (21-10.2)^2}{5-1}} = 7.5299 \end{equation*}
    Now that we have all of these, letâ€™s get down to the work of calculating the correlation. Our next step is to subtract the mean from each data value in \(x\) and \(y\text{:}\)
    ðŸ”—
    \begin{gather*} (x - \overline{x}) = \lbrace (1-5.2),(2-5.2),(5-5.2),(6-5.2),(12-5.2)\rbrace \\ = \lbrace -4.2,-3.2,-0.2,0.8,6.8\rbrace \end{gather*}
    \begin{gather*} (y - \overline{y}) = \lbrace (2-10.2),(5-10.2),(9-10.2),(14-10.2),(21-10.2)\rbrace\\ = \lbrace -8.2,-5.2,-1.2,3.8,10.8\rbrace \end{gather*}
    Now weâ€™ll multiply each corresponding pair together:
    ðŸ”—
    \begin{gather*} (x - \overline{x})*(y-\overline{y}) = \lbrace -4.2*-8.2,-3.2*-5.2,-0.2*-1.2,0.8*3.8,6.8*10.8\rbrace \\ = \lbrace 34.44, 16.64, 0.24, 3.04, 73.44\rbrace \end{gather*}
    Now weâ€™ll add up all the values:
    ðŸ”—
    \begin{equation*} 34.44 + 16.64 + 0.24 + 3.04 + 73.44 = 127.8 \end{equation*}
    And finally, we divide by \((n-1)s_x s_y\text{.}\)
    ðŸ”—
    \begin{equation*} cor(x,y) = \frac{127.8}{(5-1)*4.3244*7.5299} = 0.9811 \end{equation*}
    This is the same value we got from R above, up to 3 decimal places (the difference comes because we rounded \(s_x\) and \(s_y\))

    Subsection 2.7.3 Using Correlation

    Ok, so weâ€™ve defined this correlation, which tells us how one set of numbers is related to another. But why does it matter?
    ðŸ”—
    An old adage says that "correlation does not imply causation," meaning that just because two things are correlated doesnâ€™t mean that change to one causes the change in another. Two things could be correlated that are completely unrelated. For example, there is a strong correlation (0.9039) between the number of bee colonies in the US and the divorce rate in South Carolina:
    ðŸ”—
    Does this mean that bees are breaking up marriages in South Carolina, or that new divorcees are running out to start beekeeping? Of course not (or at least, probably not). We donâ€™t have any other reason to believe that one of these variables is causing the other. But when there is some reason why we might believe that one thing is causing another, the correlation can give us evidence to support that theory.
    ðŸ”—
    For example, because of the legacy of systemic racism and intentional exclusion of Black Americans from the financial system, we could theorize that there are fewer banks - and more payday lenders - in majority Black neighborhoods. We could use correlation to see if there is evidence for this theory - by looking at the racial makeup of a neighborhood and seeing if there is a correlation with the number of financial institutions. This doesnâ€™t prove that banks are intentionally avoiding Black neighborhoods (though other evidence may support that claim) - it just provides evidence to support that theory.
